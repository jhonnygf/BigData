Lab — Detección de Fugas de Agua

Pipeline completo: sensores IoT → Kafka → Spark Structured Streaming → modelo KMeans de detección de anomalías → alertas en tiempo real. Dataset sintético de 10 minutos, 1 lectura/segundo por sensor.

Arquitectura del sistema

CapaComponenteRolTecnología
IngestiónProductor PythonSimula 5 sensores de agua enviando JSONkafka-python
MensajeríaKafka topic sensores-aguaBuffer duradero de eventos en tiempo realConfluent Kafka 7.5
ProcesamientoSpark Streaming JobLee de Kafka, aplica modelo, emite alertasPySpark 3.5 Structured Streaming

ℹ️
¿Por qué KMeans para anomalías? Scikit-learn's IsolationForest no existe en MLlib nativo. KMeans en modo anomaly scoring es una alternativa directa en PySpark: entrenamos con datos normales y cualquier lectura cuya distancia al centroide más cercano supere un umbral se marca como anomalía.

Schema del sensor

Cada sensor envía un JSON con este formato a Kafka cada segundo:

sensor_idSTRING — "sensor-01" … "sensor-05"
ts_epoch_msLONG — epoch en milisegundos
caudal_lpmDOUBLE — litros/minuto (normal: 10-30)
presion_barDOUBLE — bar (normal: 1.5-3.5)
temp_cDOUBLE — °C (normal: 8-22)
json — ejemplo lectura normal
{"sensor_id": "sensor-02", "ts_epoch_ms": 1745395200000,
 "caudal_lpm": 18.4, "presion_bar": 2.7, "temp_c": 14.1}
json — ejemplo lectura con FUGA (caudal anormal)
{"sensor_id": "sensor-02", "ts_epoch_ms": 1745395260000,
 "caudal_lpm": 78.9, "presion_bar": 0.4, "temp_c": 14.3}

Fase 1 Generar datos de entrenamiento

Crea el fichero lab_jobs/01_generar_datos.py con el siguiente contenido:

python — lab_jobs/01_generar_datos.py
"""
Genera un CSV de lecturas NORMALES para entrenar el modelo KMeans.
10 minutos × 5 sensores × 1 lectura/s = 3 000 filas
"""
import numpy as np
import pandas as pd
import time

SENSORES    = [f"sensor-{i:02d}" for i in range(1, 6)]
DURACION_S  = 600   # 10 minutos de datos sintéticos
SALIDA      = "lab_data/entrenamiento_normal.csv"

rng = np.random.default_rng(seed=42)
filas = []

ts_base = int(time.time() * 1000) - DURACION_S * 1000

for t in range(DURACION_S):
    ts = ts_base + t * 1000
    for sid in SENSORES:
        filas.append({
            "sensor_id":   sid,
            "ts_epoch_ms": ts,
            "caudal_lpm":  rng.normal(loc=20, scale=3),     # normal: ~20 lpm
            "presion_bar": rng.normal(loc=2.5, scale=0.3),  # normal: ~2.5 bar
            "temp_c":      rng.normal(loc=15,  scale=2),    # normal: ~15 °C
        })

df = pd.DataFrame(filas)
df["caudal_lpm"]  = df["caudal_lpm"].clip(5, 40)
df["presion_bar"] = df["presion_bar"].clip(1.0, 4.0)
df["temp_c"]      = df["temp_c"].clip(5, 30)

df.to_csv(SALIDA, index=False)
print(f"Guardadas {len(df)} filas en {SALIDA}")
print(df.describe().round(2))

Fase 2 Entrenar el modelo de anomalías (KMeans)

Crea lab_jobs/02_entrenar_modelo.py:

python — lab_jobs/02_entrenar_modelo.py
"""
Entrena un KMeans con datos normales.
Estrategia de anomalía: si la distancia de una lectura al centroide más cercano
supera UMBRAL_DISTANCIA, se considera anómala (posible fuga).
"""
from pyspark.sql import SparkSession
from pyspark.ml import Pipeline
from pyspark.ml.feature import VectorAssembler, StandardScaler
from pyspark.ml.clustering import KMeans
from pyspark.sql.functions import col
import json

DATOS    = "lab_data/entrenamiento_normal.csv"
MODELO   = "lab_models/kmeans_fugas"
K        = 5   # un cluster por "estado operativo" esperado

spark = SparkSession.builder \
    .appName("EntrenarKMeans") \
    .master("local[*]") \
    .getOrCreate()
spark.sparkContext.setLogLevel("WARN")

df = spark.read.csv(DATOS, header=True, inferSchema=True)
df.show(5)

# ── Pipeline: VectorAssembler → StandardScaler → KMeans ──────────
FEATURES = ["caudal_lpm", "presion_bar", "temp_c"]

assembler = VectorAssembler(inputCols=FEATURES, outputCol="features_raw")
scaler    = StandardScaler(inputCol="features_raw", outputCol="features",
                           withMean=True, withStd=True)
kmeans    = KMeans(featuresCol="features", k=K, seed=42, maxIter=50)

pipeline = Pipeline(stages=[assembler, scaler, kmeans])
model    = pipeline.fit(df)

model.save(MODELO)
print(f"Modelo guardado en {MODELO}")

# ── Calcular umbral de anomalía ──────────────────────────────────
# Percentil 99 de las distancias en datos normales = umbral
from pyspark.ml.linalg import Vectors
import numpy as np

transformed = model.transform(df)
km_model    = model.stages[-1]
centers     = km_model.clusterCenters()

def distancia_centroide(row):
    vec        = row.features.toArray()
    cluster    = row.prediction
    centroide  = np.array(centers[cluster])
    return float(np.linalg.norm(vec - centroide))

from pyspark.sql.types import DoubleType
from pyspark.sql.functions import udf

dist_udf = udf(distancia_centroide, DoubleType())
dists = transformed.withColumn("dist", dist_udf(transformed["features", "prediction"]))

# Percentil 99 como umbral
p99 = dists.approxQuantile("dist", [0.99], 0.01)[0]
print(f"\nUmbral de anomalía (p99): {p99:.4f}")

umbral = {"umbral_distancia": p99, "features": FEATURES}
with open("lab_models/umbral.json", "w") as f:
    json.dump(umbral, f, indent=2)
print("Umbral guardado en lab_models/umbral.json")

spark.stop()

Fase 3 Job de detección en tiempo real

Este es el job principal. Lee de Kafka, aplica el modelo, y emite alertas. Créalo como lab_jobs/03_streaming_alertas.py:

python — lab_jobs/03_streaming_alertas.py
"""
Job de Structured Streaming:
  Kafka topic 'sensores-agua'
      → deserializar JSON
      → VectorAssembler + StandardScaler (del modelo guardado)
      → KMeans.transform → calcular distancia al centroide
      → Si dist > umbral  → FUGA_POSIBLE
      → Escribir a Kafka topic 'alertas-fugas' Y consola
"""
import json
import numpy as np

from pyspark.sql import SparkSession
from pyspark.ml import PipelineModel
from pyspark.sql.functions import (
    from_json, col, to_json, struct, lit,
    when, to_timestamp, udf
)
from pyspark.sql.types import (
    StructType, StructField,
    StringType, LongType, DoubleType
)

# ── Configuración ─────────────────────────────────────────────────
KAFKA_BOOTSTRAP = "localhost:9092"   # desde el host
TOPIC_IN        = "sensores-agua"
TOPIC_OUT       = "alertas-fugas"
MODELO_PATH     = "lab_models/kmeans_fugas"
UMBRAL_PATH     = "lab_models/umbral.json"
CHECKPOINT      = "/tmp/checkpoint/alertas_streaming"

# ── Cargar umbral de anomalía ─────────────────────────────────────
with open(UMBRAL_PATH) as f:
    cfg = json.load(f)
UMBRAL    = cfg["umbral_distancia"]
FEATURES  = cfg["features"]
print(f"Umbral cargado: {UMBRAL:.4f}")

# ── Spark Session ─────────────────────────────────────────────────
spark = SparkSession.builder \
    .appName("DeteccionFugas_Streaming") \
    .master("local[*]") \
    .config("spark.jars.packages",
            "org.apache.spark:spark-sql-kafka-0-10_2.12:3.5.0") \
    .getOrCreate()
spark.sparkContext.setLogLevel("WARN")

# ── Cargar modelo MLlib ────────────────────────────────────────────
pipeline_model = PipelineModel.load(MODELO_PATH)
km_model       = pipeline_model.stages[-1]
centers_bc     = spark.sparkContext.broadcast(
    [c.toArray().tolist() for c in km_model.clusterCenters()]
)

# ── Schema del mensaje Kafka ───────────────────────────────────────
sensor_schema = StructType([
    StructField("sensor_id",   StringType()),
    StructField("ts_epoch_ms", LongType()),
    StructField("caudal_lpm",  DoubleType()),
    StructField("presion_bar", DoubleType()),
    StructField("temp_c",      DoubleType()),
])

# ── UDF: distancia al centroide asignado ──────────────────────────
def dist_centroide(features_vec, cluster_idx):
    centers = centers_bc.value
    arr     = features_vec.toArray()
    centro  = np.array(centers[cluster_idx])
    return float(np.linalg.norm(arr - centro))

dist_udf = udf(dist_centroide, DoubleType())

# ── Lectura de Kafka (stream) ─────────────────────────────────────
raw_stream = spark.readStream \
    .format("kafka") \
    .option("kafka.bootstrap.servers", KAFKA_BOOTSTRAP) \
    .option("subscribe", TOPIC_IN) \
    .option("startingOffsets", "latest") \
    .option("failOnDataLoss", "false") \
    .load()

parsed = raw_stream \
    .selectExpr("CAST(value AS STRING) AS json_str",
                "timestamp AS kafka_ts") \
    .select(
        from_json(col("json_str"), sensor_schema).alias("d"),
        col("kafka_ts")
    ) \
    .select("d.*", "kafka_ts") \
    .withColumn("event_time", to_timestamp(col("ts_epoch_ms") / 1000))

# ── Aplicar pipeline MLlib ────────────────────────────────────────
# (VectorAssembler + StandardScaler ya están en el pipeline guardado)
# Sólo necesitamos aplicar las primeras 2 etapas (no el KMeans)
scaler_model    = pipeline_model.stages[1]   # StandardScalerModel
assembler_stage = pipeline_model.stages[0]   # VectorAssembler

assembled  = assembler_stage.transform(parsed)
scaled     = scaler_model.transform(assembled)
predicted  = km_model.transform(scaled)

with_dist = predicted.withColumn(
    "distancia",
    dist_udf(col("features"), col("prediction"))
)

# ── Clasificar: normal vs anomalía ────────────────────────────────
UMBRAL_LIT = float(UMBRAL)

with_alerta = with_dist.withColumn(
    "alerta",
    when(col("distancia") > UMBRAL_LIT, lit("FUGA_POSIBLE"))
    .otherwise(lit("OK"))
)

alertas_only = with_alerta.filter(col("alerta") == "FUGA_POSIBLE")

# ── Sink 1: consola (para ver en tiempo real en terminal) ─────────
q_consola = with_alerta \
    .select("sensor_id", "event_time", "caudal_lpm",
            "presion_bar", "temp_c", "distancia", "alerta") \
    .writeStream \
    .outputMode("append") \
    .format("console") \
    .option("truncate", False) \
    .trigger(processingTime="5 seconds") \
    .start()

# ── Sink 2: Kafka topic alertas-fugas ─────────────────────────────
q_kafka = alertas_only \
    .select(
        col("sensor_id").alias("key"),
        to_json(struct(
            "sensor_id", "event_time",
            "caudal_lpm", "presion_bar", "temp_c",
            "distancia", "alerta"
        )).alias("value")
    ) \
    .writeStream \
    .outputMode("append") \
    .format("kafka") \
    .option("kafka.bootstrap.servers", KAFKA_BOOTSTRAP) \
    .option("topic", TOPIC_OUT) \
    .option("checkpointLocation", CHECKPOINT + "/kafka") \
    .trigger(processingTime="5 seconds") \
    .start()

print("━" * 60)
print("  Streaming activo. Esperando datos en 'sensores-agua'...")
print("  Spark UI → http://localhost:4040")
print("  Para parar: Ctrl+C")
print("━" * 60)

spark.streams.awaitAnyTermination()

python — lab_jobs/04_productor_sensores.py
"""
Simulador de sensores de agua.
Envía una lectura por sensor por segundo durante DURACION_S segundos.

Parámetros opcionales (modifica las variables de abajo):
  DURACION_S     — duración total en segundos (default: 600 = 10 min)
  FUGAS_EN_S     — lista de instantes en segundos donde se inyectan fugas
                   (ej: [120, 300, 480] → fugas al min 2, 5 y 8)
  SENSORES_FUGA  — qué sensores se ven afectados en esas fugas
"""
import json
import time
import numpy as np
from kafka import KafkaProducer

# ── Configuración ─────────────────────────────────────────────────
KAFKA_BOOTSTRAP = "localhost:9092"
TOPIC           = "sensores-agua"
DURACION_S      = 600                  # 10 minutos
SENSORES        = [f"sensor-{i:02d}" for i in range(1, 6)]

# Instantes (segundos desde el inicio) donde hay una fuga
FUGAS_EN_S      = [120, 240, 360, 480]
SENSORES_FUGA   = {"sensor-02", "sensor-04"}   # qué sensores fallan

# ── Productor Kafka ────────────────────────────────────────────────
producer = KafkaProducer(
    bootstrap_servers=KAFKA_BOOTSTRAP,
    value_serializer=lambda v: json.dumps(v).encode("utf-8"),
    key_serializer=lambda k: k.encode("utf-8"),
    acks="all",
)

rng = np.random.default_rng()

print(f"Enviando datos a topic '{TOPIC}' durante {DURACION_S}s...")
print(f"Fugas programadas en segundos: {FUGAS_EN_S}")
print("Ctrl+C para parar antes de tiempo\n")

for t in range(DURACION_S):
    fuga_activa = t in FUGAS_EN_S

    for sid in SENSORES:
        is_broken = fuga_activa and sid in SENSORES_FUGA

        if is_broken:
            # Simular fuga: caudal disparado, presión caída
            caudal  = rng.uniform(60, 100)
            presion = rng.uniform(0.1, 0.6)
            temp    = rng.normal(15, 2)
        else:
            # Condiciones normales
            caudal  = rng.normal(20, 3)
            presion = rng.normal(2.5, 0.3)
            temp    = rng.normal(15, 2)

        msg = {
            "sensor_id":   sid,
            "ts_epoch_ms": int(time.time() * 1000),
            "caudal_lpm":  round(float(caudal),  2),
            "presion_bar": round(float(presion), 3),
            "temp_c":      round(float(temp),    2),
        }

        producer.send(TOPIC, key=sid, value=msg)

        if is_broken:
            print(f"  💧 FUGA inyectada → {sid}: caudal={caudal:.1f} lpm, "
                  f"presion={presion:.2f} bar")

    producer.flush()

    # Indicador de progreso cada 30 segundos
    if t % 30 == 0:
        mins = t // 60; segs = t % 60
        print(f"  t={mins:02d}:{segs:02d} — lecturas enviadas: {(t+1)*len(SENSORES)}")

    time.sleep(1)   # 1 lectura/s por sensor

producer.close()
print("\nProductor finalizado.")
