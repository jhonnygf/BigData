Apache Spark con PySpark

De cero a producción: arquitectura distribuida, programación funcional, SQL, Machine Learning y Streaming en tiempo real. Con un lab final de detección de fugas de agua.

📚 Mapa del Curso

Progresión guiada desde los fundamentos del procesamiento distribuido hasta un sistema de detección de anomalías en tiempo real con ML. Cada módulo construye sobre el anterior.

🏗️

M1 — Fundamentos & Docker

¿Qué es Spark y por qué existe? Arquitectura Driver/Executor, DAG, particiones, modos de despliegue. Entorno completo con Docker Compose en minutos.
λ

M2 — Programación Funcional

El paradigma que subyace a Spark: funciones puras, inmutabilidad, map, filter, reduce, lambdas, closures y composición. En Python puro antes de tocar Spark.
🧱

M3 — RDDs & DataFrames

Los bloques fundamentales. RDD: colección distribuida e inmutable. DataFrame: RDD con esquema. Transformaciones lazy vs acciones eager. Linaje y tolerancia a fallos.
🗄️

M4 — Spark SQL

Catalyst Optimizer, Tungsten engine, esquemas StructType, joins distribuidos (broadcast vs shuffle), window functions, lectura/escritura de Parquet, CSV y JSON.
🤖

M5 — MLlib: Machine Learning

Pipelines ML, transformers y estimators, VectorAssembler, clasificación (RandomForest, GBT), regresión, clustering (KMeans), evaluación y persistencia de modelos.
🌊

M6 — Structured Streaming

Micro-batches vs continuous, sources (Kafka, socket, ficheros), sinks (consola, Parquet, Kafka), triggers, watermarks para datos tardíos y stateful operations.
💧

Lab — Detección de Fugas de Agua

Pipeline completo: sensores IoT → Kafka → Spark Streaming → modelo IsolationForest → alertas en tiempo real. Dataset sintético de 10 minutos, 1 lectura/segundo.
💡
Prerrequisitos: Python básico (listas, funciones, clases) y conceptos de Docker (imágenes, contenedores, puertos). No se necesita experiencia previa con Spark ni sistemas distribuidos.
